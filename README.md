# common-room-reservation
At my residence, people had been sharing around edited text messages to communicate reservations for the room, so I decided to make a website with synchronisation and visualisations for when the room will be booked out.

There's two parts to this project, the form, and the visualisation. The form asks for the following information:
1. Username
2. Date of reservation
3. Optionally, the start and end time of the reservation, which is set to 18:00-23:00 by default.

Yet to complete:
The visualisation is simply the month with colours and bars to indicate which days/times are taken. I might go with a calendar approach or some sort of more abstract visualisation for this.
